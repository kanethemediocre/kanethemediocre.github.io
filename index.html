Bling Hustle

Bling Hustle is a top-down arcade action space fantasy capitalism simulator. Bling Hustle features procedural content generation, Newtonian physics, 20 unique weapons and deep upgrade mechanics for both your ship and it's weapons. Bling Hustle is a free open source project developed in javascript using the HTML Canvas. Mouse and keyboard are required, 1080p or higher resolution and fullscreen mode (hit the F key) are highly recommended

How to play:

Mouse controls:

Tapping the left mouse button fires your current weapon. A few weapons will require you to hold down the left mouse button.
The mouse wheel will scroll between available weapons.
Tapping the right mouse button fires thruster pulses, but the options menu (O key) has an option to use continuous thrusters instead.
If you're using continuous thrusters, hold down the right mouse button to fire your thrusters, but beware that you cannot fire weapons while thrusting.

Keyboard controls:

F key enables fullscreen mode, and is highly recommended to avoid accidental browser actions. Esc key exits fullscreen.
0 thru 9 keys will select weapons 0 thru 9. Shift + a number key will select weapons 10-19
Q key shows the quick planetary menu
E key shows the Engineering / Educational menu (WIP)
T key will lock on to current target, or release current target lock
O key shows the options menu.
P key cycles through probe (weapon 0) modes
A toggles autopilot
S toggles starfield visual effect
D cycles through diagnostic displays
F enables fullscreen mode
J cycles Journal / Jobs display
K saves current game (you must use the GUI to download it!)
L loads the game selected in the GUI
Z toggles Autobrake
C cycles through Omnicompass modes
V cycles through Virtual Key modes
B key uses a booster
N cycles through Navigation modes
M cycles through Map modes
< and > cycle through navigation targets
Arrow keys cycle through available weapons while flying, and navigates shop menu while docked
Enter key selects shop items while docked
Delete key activates cheat mode
Insert key gives all weapons but no upgrades (cheat mode only)
End key gives 10,000 bling (cheat mode only)
